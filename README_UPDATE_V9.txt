VERSI UPDATE: THS Analytics Online v9 Smart Update

Yang ditambahkan pada versi ini:
1. Data lama tidak dihapus saat import.
2. Import menggunakan Smart Update:
   - Data baru otomatis ditambahkan.
   - Data lama yang berubah otomatis diperbarui.
   - Data yang sama otomatis dilewati supaya tidak dobel.
   - Duplikat di dalam file Excel yang sama dirapikan.
3. Fitur v8 tetap dipertahankan:
   - Warna lead/status.
   - Grafik dashboard.
   - Klik grafik untuk membuka detail customer.
   - Export Excel berisi Dashboard, Detail Filter, dan Semua Data.
4. Bulan berjalan sekarang juga mode smart update, bukan replace/hapus otomatis.

Kunci pembanding anti-dobel:
Repair Date + Police No + No Rangka + Repair Type + SA + KM

Catatan:
- Jangan hapus database Supabase jika hanya ingin update data.
- Cukup upload file terbaru, sistem akan cek mana yang baru, mana yang berubah, dan mana yang sama.
- Tombol Reset tetap ada, tetapi hanya digunakan kalau benar-benar ingin mengosongkan data.
